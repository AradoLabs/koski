TOR-tiedonsiirtoprotokolla

Tässä kuvataan TOR-järjestelmän tiedonsiirrossa käytettävä JSON-formaatti.

JSON Schema:

  • tor-oppija-schema.json
  • visualisointi

JSON Schema validaattori netissä: jsonschemavalidator.net. Voit kokeilla laittaa sinne meidän scheman ja esimerkin.
